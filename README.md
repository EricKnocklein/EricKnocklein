## Projects
- **[Stochastic Particle Simulation](https://github.com/EricKnocklein/stochastish-gravity)**: An exploration of how local differences in the speed of updating particles gives a global force similar to gravity in a particle simulation. Plushed as an [interactive article](https://ericknock.dev/gravity/).
- **[CSS Book Animation](https://github.com/EricKnocklein/animated_book_css)**: Uses 3D transforms to create an openable book that displays arbitrary HTML content. You can find an example of this component being used in practice on my [author's website](https://eklittlehill.com/), where I use the component as a mock-up for a short book I wrote.
- **[World Anvil Spell Check Override Chrome Extension](https://github.com/EricKnocklein/WaSpellcheck)**: I use World Anvil a lor for my personal projects, and they had an update that remove spell-checking from one of their editors, so I wrote a simple chrome extension to put it back.
- **[Portfolio Template](https://github.com/EricKnocklein/portfolio_template)**: When I was creating my personal website, I decided to publish the React components so that others could easily create a similar portfolio site.
- **[Wiki Reading Order](https://github.com/EricKnocklein/Algorithmic-Reading-Order-2)**: Python code to organize a wiki into a reading order using an adapted topological sort algorithm. There are also [2 YouTube videos](https://ericknock.dev/projects/graph%20analysis) to go along with this repo and part 1 of this repo.
